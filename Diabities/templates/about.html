About This Project

This Flask-based web application uses a machine learning model to predict whether a patient is likely to have diabetes based on medical inputs.

  • preg: Number of pregnancies
  • plas: Plasma glucose concentration
  • pres: Diastolic blood pressure
  • skin: Triceps skin fold thickness
  • insu: 2-Hour serum insulin
  • mass: Body mass index (BMI)
  • pedi: Diabetes pedigree function
  • age: Age in years
